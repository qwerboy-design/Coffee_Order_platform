# Cursor 快速呼叫模型使用次數查詢指南

## 方法 1: 通過 Cursor 設定頁面（推薦）

最直接的方式是通過 Cursor 的官方設定頁面查看使用統計：

1. **打開瀏覽器**，訪問：https://cursor.com/settings
2. **登入您的 Cursor 帳戶**
3. **查看使用情況**：在設定頁面中，您會看到：
   - 快速 (Fast) premium 模型的使用次數
   - 慢速 (Slow) premium 模型的使用次數
   - 當月剩餘使用次數

## 方法 2: 在 Cursor 編輯器中查看

### 通過狀態欄查看
- 查看 Cursor 底部狀態欄，通常會顯示使用相關信息

### 通過命令面板查看
1. 按 `Ctrl+Shift+P` (Windows/Linux) 或 `Cmd+Shift+P` (Mac)
2. 輸入 `Usage` 或 `Quota`
3. 查看相關命令和統計信息

### 通過設定界面查看
1. 按 `Ctrl+,` (Windows/Linux) 或 `Cmd+,` (Mac) 打開設定
2. 搜尋 "usage" 或 "quota"
3. 查看相關的使用統計設定

## 訂閱計劃與使用限制

根據 Cursor 的訂閱計劃，每月的使用次數限制如下：

### 愛好者計劃 (Hobby Plan)
- **慢速 premium 模型**：每月 50 次
- **快速 premium 模型**：0 次

### 專業版計劃 (Pro Plan)
- **快速 premium 模型**：每月 500 次
- **慢速 premium 模型**：無限制

### 企業版計劃 (Business Plan)
- **快速 premium 模型**：每月 500 次
- **慢速 premium 模型**：無限制
- 額外的企業功能

## 重要說明

1. **使用次數重置**：使用次數會根據您的訂閱開始日期每月重置
2. **按使用量計費**：如果需要更多快速 premium 模型使用次數，可以在設定頁面中選擇按使用量計費，或升級訂閱計劃
3. **雲端存儲**：使用統計通常存儲在 Cursor 的雲端伺服器上，本地文件系統中可能無法直接查詢完整的統計信息

## 本地檢查腳本

如果需要在本地檢查相關文件，可以執行：

```powershell
.\scripts\check-cursor-usage.ps1
```

這個腳本會搜尋：
- Cursor 配置目錄中的使用統計文件
- 工作區存儲中的狀態文件
- 日誌文件中的相關信息

**注意**：本地檢查可能無法獲得完整的使用統計，建議使用官方設定頁面查看。
